# ClinicVet.PetCare.Api

A API ClinicVet possibilita o gerenciamento de tutores, pets e seus agendamentos de procedimentos veterinários, 
disponibilizando endpoints que garantem a integridade e a consistência dos dados.

## Requisitos Funcionais

1.Gestão de Tutores

- Criar, listar, atualizar informações referentes aos tutores.

2.Gestão de Pets

- Registrar, listar, atualizar informações referentes aos pets.

3.Gestão de Agendamentos

- Agendar, listar, atualizar informações referente aos procedimentos veterinários.

## Tecnologias Utilizadas

**Backend:**

- .NET 8
- C# 12
- ASP.NET Core

**DataBase:**

- Oracle Database

## Atributos de Qualidade

- Simplicidade de operação e manutenção;

- Performance otimizada no acesso a dados;

- Confiabilidade e consistência das informações gerenciadas;
